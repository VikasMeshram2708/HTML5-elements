<abbr> Element

The <abbr> element represents an abbreviation or acronym.

Element Attributes:
title
A text description of the abbreviation or acronym.
Example Attribute Values:
  • title="World Wide Web Consortium"
  • title="Hypertext Markup Language"
  • title="Cascading Style Sheets"
Description of Attributes:
  • title: This attribute is required. It specifies the text description of the abbreviation or acronym. When the user hovers over or focuses on the element, the description will be displayed as a tooltip.

Example Usage:

Below is an example of how to use the <abbr> element:

      
        <p>
          The W3C</abbr> develops web standards.
        </p>
